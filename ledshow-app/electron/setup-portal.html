Configureer en download de agent software voor je externe computer.

Let op: Voor installatie zonder internet moet je Node.js en MPV installers reeds gedownload hebben.

De bovenstaande configuratie wordt direct in het start.bat bestand geplaatst als omgevingsvariabelen.

3. Installatie Instructies (Foolproof)

  1. Stap 1: Node.js installeren
    Download en installeer Node.js (LTS v22). Dit is nodig om de agent software te kunnen draaien.
  2. Stap 2: MPV Player installeren
    Download de shinchiro build van MPV Player (64-bit).
    Tip: Pak de zip uit en voeg de map toe aan je Windows 'Path' OF plaats mpv.exe direct in de agent map na het uitpakken.
  3. Stap 3: Agent uitpakken
    Pak de ledshow-agent.zip uit naar een map op je computer (bijvoorbeeld C:\LedShowAgent\).
  4. Stap 4: Start script plaatsen
    Download het start.bat bestand hierboven en plaats deze direct in de map waar ook package.json en dist/ staan.
  5. Stap 5: Starten!
    Dubbelklik op start.bat. Er opent een zwart venster. De eerste keer kan Windows vragen om netwerktoegang; sta dit toe.
  6. Stap 6: Controle
    Ga op de computer waar de agent draait naar http://localhost:3000 om de status te zien.
